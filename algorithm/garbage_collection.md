## 垃圾回收
* 引用计数法，记录对象被引用的次数，清除引用次数为零的对象。
* 标记清除法，从根节点出发，标记未被引用的对象，定期清除。

## 引用计数法
```js
function count() {
  const a = {};// 应用类型变量，引用次数为0
  let b = a;   // a的引用次数变为1
  let c = a;   // a的引用次数变为2
  b = {};      // b不再引用a a引用次数减1
  c = null;    // c不再引用a a引用次数为0，垃圾回收
}
```
