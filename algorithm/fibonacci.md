## 斐波那契数列
`1,1,2,3,5,8,13...`

`f(1)=1`

`f(2)=1`

`f(n)=f(n-1)+f(n-2)`

## 递归方式
```js
function fib(n) {
  if (n <= 2) {
    return 1;
  }
  
  return fib(n - 2) + fib(n - 1);
}
```
缺点，重复计算
`f(5) = f(4) + f(3)`
`f(4) = f(3) + f(2)`
`f(3)`会计算两次

## 尾递归优化
当n很大，每一层函数执行都会创建参数，到时内存溢出，浏览器卡死。
解决方式，在函数参数中计算值
```js
function fib(n, n1 = 1, n2 = 1) {
  if (n <= 2) {
    return n2;
  }

  return fib(n - 1, n2, n1 + n2);
}
```
